# Yoga Rest API

## Requirements
1. Java - 1.8.x
2. Maven - 3.x.x
3. Mysql - 5.x.x

** Mysql database
db_yoga.sql

** Run
mvn spring-boot:run (<http://localhost:8080>)

## Rest APIs
    GET /api/postures
    POST /api/postures
    GET /api/postures/{id}
    PUT /api/postures/{id}
    DELETE /api/postures/{id}
